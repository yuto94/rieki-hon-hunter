# 利益本ハンター

本せどり仕入れをテーマにした、ブラウザで遊べる60秒ミニゲームです。

プレイヤーは古本屋で本せどらーになり、表示される本の「仕入れ価格」「販売価格」「手数料」を見て、利益が出そうな本を時間内に仕入れていきます。ゲーム中は利益額を表示せず、暗算で判断する楽しさを重視しています。

## ゲーム概要

- タイトル: 利益本ハンター
- ジャンル: 本せどり仕入れ判断ゲーム
- プレイ時間: 60秒
- 目標利益: 10,000円
- 対応環境: PC / スマホのブラウザ
- 使用技術: HTML / CSS / JavaScript

## 現在の完成状態 v1.0

- TOP画面にタイトルロゴとポップな背景を表示
- ゲーム開始前にレトロRPG風の導入テロップを表示
- 毎回ランダムに30冊の本棚を生成
- ジャンル、タイトル、表紙テンプレート、利益ランク、価格生成ルールを設定として分離
- 高利益、中利益、低利益、赤字のバランスを調整
- ゲーム中は利益額を表示せず、仕入れ価格・販売価格・手数料だけを表示
- 結果画面で見込み利益、仕入れ冊数、目標利益、達成率を表示
- 仕入れた本ごとの利益一覧を結果画面で表示
- 結果に応じたBrain教材紹介カードを表示
- 将来の「本せどりタウン」連携用に結果データを生成

## 遊び方

`index.html` をブラウザで開くと遊べます。

サーバーにアップロードする場合は、このフォルダ内のファイル一式を同じ階層構造のまま公開ディレクトリへ配置してください。
